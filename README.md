# Nekozane (VoidChan)

- A simple helper Discord bot for [VoidChan](https://github.com/reinacchi/voidchan).

**Mugi Nekozane** (猫実 麦) is the female protagonist of the romantic comedy manga series **Learning to Love My Cat-like Classmate** (Japanese: Tonari no Neko to Koi Shirazu), written and illustrated by Akinoko.

This project is exclusively designed for the VoidChan ecosystem and should not be utilised for any personal purposes.
